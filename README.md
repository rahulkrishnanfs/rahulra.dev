# Rahulkrishnan R A — Personal Branding Website

A modern, responsive personal branding website built with **React** (Vite) and **Bootstrap** (react-bootstrap + bootstrap-icons).

🏅 Featuring the **OpenUK 2026 Honour**.

## Tech stack

- React 19 + Vite
- Bootstrap 5 / react-bootstrap
- Bootstrap Icons
- Custom dark theme inspired by OpenUK branding

## Sections

- **Hero** — name, role, ambassador pills, OpenUK honour badge & social links
- **About** — bio + quick stats
- **Specialities** — what I do (Cloud, CI/CD, Kubernetes, Microservices, IaC, Monitoring, GenAI, Dev)
- **Skills** — languages & tools (Golang, Python, JS, Kubernetes, Docker, Terraform, Ansible, Jenkins, ELK…)
- **Speaking & Community** — talks and open-source involvement
- **Contact** — email + socials

## Getting started

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build into /dist
npm run preview  # preview the production build
```

## Customising

- Edit `src/data.js` to update text, links, email, skills and talk topics.
- Theme colors live as CSS variables at the top of `src/index.css`.
- Replace `src/assets/openuk-honour-2026.png` to swap the hero image.

> Update the placeholder social URLs and `email` in `src/data.js` with your real links.
